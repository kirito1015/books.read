半小时漫画论语

# 一、论语是怎么来的

《论语》，俗称“孔子的朋友圈”，主要记载了孔子和他弟子们日常说的话、做的事。

## 1.1 孔子是谁 
孔子出生在春秋时期的鲁国（今山东）。孔子父亲劳来得子生了他。可孔子才三岁，父亲就去世了。而孔子十七岁时母亲也去世了。  
面对人生的苦难，孔子艰苦奋斗、自力更生，在社会底层从事过多种职业。但他更看重追求学问的光明大道，无时无刻不在学习。
> ***子曰：“十室之邑，必有忠信如丘者焉，不如丘之好学也。”***  
> 孔子说：“就是只有十户人家的小地方，也一定有像我这样忠诚信实的人，只是赶不上我喜欢学问的程度罢了。”

## 1.2 孔子都做了些什么？
早年开办学校，教书育人；  
中年被职场排挤，离开鲁国，周游列国，四处求职，推广思想；  
晚年重回故土，整理书籍，传承文化。

### 1.2.1 开办学校，教书育人
春秋时期，只有贵族能读书，一般人没有机会上学。但孔子不一样，只有有心求学，都可以来上课。
> ***子曰：“有教无类。”***  
> 孔子说：“人人我都教育，没有（贫富、地域、身份等方面）的区别。”

孔老师大半辈子教书育人，兢兢业业，几乎全年无休。他给自己定的目标是：学而不厌，诲人不倦。
> ***子曰：“默而识(zhi)之，学而不厌，诲人不倦，何有于我哉？”***  
> （把所见所闻）默默地记在心里，努力学习而不感到满足，教导别人而不疲倦，这些事情我做到了哪些呢？
