# 2. 丰富的经历 = 更大的大脑吗
> 某些特定的经验是否会引起大脑形态发生改变?
1. 方法  
将大鼠分成3组，标准、贫乏环境、丰富环境培养一段时间后对大脑进行测量、称重和分析。
2. 结果  
丰富环境下大脑皮层更厚、更重，神经元也更大。
3. 相关研究  
在生活中，我们应该对大脑老化持更加乐观的态度····主要因素是刺激，神经细胞因刺激而存在。好奇心是一个关键因素。如果一个人在一生中始终充满好奇心，这将刺激他的神经组织和大脑皮层。经常用脑的人大脑不易老化。
